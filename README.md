# Iron Coaching | Muscle Building Academy 🏋️‍♂️

A clean, responsive, single-page website built to educate users on the science of building muscle. This project serves as a landing page and educational hub for a fitness coaching brand, emphasizing raw strength, hypertrophy, and progressive overload.

## 🚀 Features

*   **Modern Hero Section:** A striking landing area with a clear call-to-action.
*   **The Triangle of Gains:** An educational section breaking down the three pillars of muscle growth: Training (Stimulus), Nutrition (Fuel), and Recovery (Adaptation).
*   **Workout Routines Module:** A clean, easy-to-read table layout detailing a 3-day full-body beginner hypertrophy split (Exercises, Sets, Reps, and Rest periods).
*   **Fully Responsive:** Custom CSS ensures the layout looks great on both desktop monitors and mobile devices.

## 🛠️ Built With

*   **HTML5** - For semantic structure and content.
*   **CSS3** - For styling, layout, and mobile responsiveness (no external frameworks used).
